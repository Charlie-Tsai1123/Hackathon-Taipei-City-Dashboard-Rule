工作坊一些注意事項
1. 主題有六個，比賽當天會從六個選出三個主題進行比賽
2. 至少四個雙北組件 (資訊局局長建議針對同一個主題) ，至少一個雙北組件含地圖圖層
3. 指定題必須完成，自選題是加分項目
4. 建議 csv 匯入資料
5. 之前團隊有使用 navicat 或 deliver 軟體進行資料庫備份
